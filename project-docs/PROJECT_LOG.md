## 2026-05-04

- Определено, что проект является React SPA на Vite.
- Проверена точка входа: `index.html` -> `src/main.jsx` -> `src/App.jsx`.
- Проверены основные маршруты: `/`, `/try`, `/gallery`, `/my`, `/login`, `/auth/callback`, `/cabinet`.
- Определено, что backend в репозитории отсутствует: используются Supabase и внешний webhook генерации.

- Проверен кастомный домен: `public/CNAME` содержит `app.primerkakoles.ru`.

### Проверенные Риски

- При отсутствии `VITE_SUPABASE_URL` или `VITE_SUPABASE_ANON_KEY` код пишет ошибку в console, но все равно вызывает `createClient`.
- При отсутствии `VITE_WEBHOOK_URL` генерация на `/try` упадет на `fetch`.
- В `PhotoUpload` при замене файла может оставаться старый object URL, если файл заменили без очистки.
- `GalleryPage` и `MyGenerationsPage` дублируют preview/gallery-логику.
- `vercel.json.bak` выглядит как остаток после миграции с Vercel.
- Локальная проверка `npm run build` в Codex-среде не запускалась, потому что `npm` не был доступен и `node_modules` отсутствовал.

## Решения

- `.env` не должен храниться в git.
- Реальные значения env должны быть локально в `.env` и в GitHub Secrets для GitHub Pages build.
- `VITE_SUPABASE_ANON_KEY` считается публичным frontend-ключом; безопасность должна обеспечиваться RLS Supabase и backend/webhook-проверками.
- Supabase `service_role` ключ нельзя использовать во frontend.
- `AI_CONTEXT.md` держим как стабильную карту, `PROJECT_LOG.md` как журнал, `ROADMAP.md` как план работ.
- Основной web-вход переводится на email/password, SMS не используется из-за высокой стоимости.
- Восстановление пароля пока делается через поддержку; полноценный password reset остается будущей задачей.
- На сайте новые пользователи не получают бесплатную web-примерку; тестовая примерка предлагается через Telegram-бота.
- Кнопка выхода убрана из видимой зоны кабинета и перенесена в подвал в раздел `Личный кабинет`.
- Ссылка `Поддержка` перенесена из навигации подвала в раздел `Контакты`.
- Ссылка на тестовую примерку в кабинете оформлена как отдельная кнопка `Тестовая примерка в Telegram-боте`.
- Личный кабинет сделан более навигационным: в основном блоке всегда видны `Примерить диски`, `Купить генерации`, `Мои примерки`.
- Ссылка на Telegram-бота вынесена из основного блока в отдельную синюю карточку ниже.
